Standalone Playtest Test Page

Testing Instructions:

  1. The standalone playtest should load below
  2. Check the browser console for any errors
  3. Try selecting a recipe and playing it
  4. Verify that:
    • Recipes load in the list
    • Search functionality works
    • Vessels appear properly spaced when game starts
    • Drag and drop works
    • Combinations create new vessels
    • Win condition triggers properly

Common Issues:

  • Vessels stacked: Check console for CSS/positioning errors
  • Can't combine: Check console logs for combination attempts
  • No recipes: Check Supabase connection and credentials
