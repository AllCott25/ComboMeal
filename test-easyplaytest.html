Easy Playtest Test Page

Open the browser console to see debug output.

Steps to test:

  1. Open easyplaytest.html
  2. Select a recipe date from the dropdown
  3. Click "Load Recipe"
  4. The game should start with the selected recipe
  5. Complete the recipe and click "Cook!" again
  6. The same recipe should load (not today's recipe)
  7. There should be no duplicate vessels

Expected Console Output:

[EasyPlaytest] Returning selected date: [your-selected-date]
[EasyPlaytest] Loading recipe data for selected date: [your-selected-date]
[EasyPlaytest] Clearing [N] existing vessels (if returning to title)
[EasyPlaytest] Fetching recipe for selected date: [your-selected-date]

Common Issues Fixed:

  • ✓ Today's recipe loading instead of selected recipe
  • ✓ Duplicate vessels when returning to title screen
  • ✓ Early recipe fetching before date selection
  • ✓ Completion check interfering with playtest mode
